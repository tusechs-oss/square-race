# Plugin Reloader

Enable or disable plugins from within the editor main screen.

[**Download**](https://github.com/kenyoni-software/godot-addons/releases/tag/latest)

## Compatibility

| Godot | Version       |
| ----- | ------------- |
| 4.6   | >= 1.0.0      |
| 4.5   | 1.0.0 - 1.2.2 |
| 4.4   | 1.0.0 - 1.2.2 |
| 4.3   | 1.0.0 - 1.2.2 |
| 4.2   | 1.0.0 - 1.2.2 |
